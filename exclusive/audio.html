# 🎚️ 音频编辑

# Ardour

Ardouropen in new window 是一款自由开源的让你可以在 Linux 上录音、编辑和混音的软件。

安装 Ardouropen in new windowcommunity / aur：

sudo pacman -S ardour

yay -S ardour-git

# Kwave

Kwaveopen in new window 是 KDE 开发的一款自由开源的音频编辑软件，可以录制、播放、导入和编辑许多音频文件，包括多声道文件。

安装 Kwaveopen in new windowextra / aur：

sudo pacman -S kwave

yay -S kwave-git

# Audacity

Audacityopen in new window 是一款跨平台的自由开源音频编辑软件，用于录音和编辑音频。2021 年 4 月 30 日，Audacity 被 Muse Groupopen in new window 收购。

安装 Audacityopen in new windowcommunity / aur：

sudo pacman -S audacity

yay -S audacity-git

ℹ️ 提示

Audacity 被 Muse Group 收购后，根据 相关报道open in new window，其隐私政策的变化显示该工具正在收集其用户的数据并与其它公司共享。

在各大开源社区中出现了许多愤怒的回应，GitHubopen in new window 和 Redditopen in new window 上的用户纷纷呼吁尽快派生（Fork）该软件（已经有了）。

若对此有关切，可以使用派生的版本 audiocity-gitopen in new windowaur（功能没有改动）：

yay -S audiocity-git

或者 audaciumopen in new windowaur（添加了新特性）：

yay -S audacium-git

# Mixxx

Mixxxopen in new window 集成了 DJ 使用数字音乐文件进行创造性现场混音所需的工具。

安装 Mixxxopen in new windowcommunity / aur：

sudo pacman -S mixxx

yay -S mixxx-git

# 🎼 编曲软件

# LMMS

LMMSopen in new window 是免费自由开源并且跨平台的作曲工具。通过制作旋律和节拍，合成和混合音频，安排音频片段等方法创作音乐。

安装 LMMSopen in new windowcommunity / aur：

sudo pacman -S lmms

yay -S lmms-git

# MuseScore

MuseScoreopen in new window 是 Muse Group 开发的一个跨平台的自由开源制谱软件。

安装 MuseScoreopen in new windowcommunity / cn / aur：

sudo pacman -S musescore

sudo pacman -S musescore-git

yay -S aur/musescore-git

# 🎙️ 变声器

# Lyrebird

Lyrebirdopen in new window是 自由开源且简单强大的 Linux 变声器，用 GTK 3 编写。

安装 Lyrebirdopen in new windowaur：

yay -S lyrebird
