# 🗿 建模渲染与动画制作

# 斑斓（Blender）

Blenderopen in new window 是一款开源免费的三维图形图像软件，提供从建模、动画、材质、渲染、到音频处理、视频剪辑等一系列动画短片制作解决方案。

安装 Blenderopen in new windowcommunity / aur：

sudo pacman -S blender

yay -S blender-git

# Sweet Home 3D

Sweet Home 3Dopen in new window 是一款免费的家装辅助设计软件。它能帮您通过二维的家居平面图来设计和布置您的家具，还可以用三维的视角浏览整个装修布局的全貌。

安装 Sweet Home 3Dopen in new window：

sudo pacman -S sweethome3d

# Synfig Studio

Synfig Studioopen in new window 是一款免费开源的 2D 动画软件，用于使用矢量和位图图稿创建电影质量的动画。

安装 Synfig Studioopen in new windowcommunity / aur：

sudo pacman -S synfigstudio

yay -S synfigstudio-git

# Linux Stopmotion

Linux Stopmotionopen in new window 是一个开源免费的应用程序，用于创建定格动画。

安装 Linux Stopmotionopen in new windowaur：

yay -S linuxstopmotion-git

# FontForge

FontForgeopen in new window 是一款自由开源、全功能、支持所有通用字体格式的字体编辑器。

安装 FontForgeopen in new windowextra / aur：

sudo pacman -S fontforge

yay -S fontforge-git
