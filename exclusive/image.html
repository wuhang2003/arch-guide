# 🖼️ 位图编辑

# GIMP

GIMPopen in new window 是一个自由开源的位图图像编辑器。其对标 Adobeopen in new window 的 Photoshopopen in new windowEULA。

安装 GIMPopen in new windowextra / cn / aur：

sudo pacman -S gimp

sudo pacman -S gimp-git

yay -S aur/gimp-git

# Aseprite

Asepriteopen in new window 是一款像素艺术绘制工具。

安装 Asepriteopen in new windowEULA / aur：

yay -S aseprite

yay -S aseprite-bin

yay -S aseprite-git

# RawTherapee

RawTherapeeopen in new window 是一个自由开源的跨平台 RAW 格式图像处理程序。

安装 RawTherapeeopen in new windowcommunity / aur：

sudo pacman -S rawtherapee

yay -S rawtherapee-git

# KolourPaint

KolourPaint 是 KDE 开发的一个简单易用的自由开源的绘图程序（类似 Windows 自带的画图），可以创建点阵图像。可用来对图片进行润色和简单编辑。

安装 KolourPaintopen in new windowextra / aur：

sudo pacman -S kolourpaint

yay -S kolourpaint-git

# 🖼️ 矢量图编辑

# Inkscape

Inkscape 是自由开源的矢量图形编辑器。该软件的开发目标是成为强大的绘图软件，且能完全遵循与支持 XML、SVG 及 CSS 等开放性的标准格式。

安装 Inkscapeopen in new windowextra / aur：

sudo pacman -S inkscape

yay -S inkscape-git

# 📑 相关资料：什么是矢量图形？

矢量图形是计算机图形学中用点、直线或者多边形等基于数学方程的几何图元表示的图像。

矢量图形与使用像素表示图像的位图不同。任意放大矢量图形，不会丢失细节或影响清晰度。

# 🎨 绘画

# Krita

Kritaopen in new window 是一个自由开源的位图图像编辑器。它针对数字绘画和动画创作特化，提供包括低干扰用户界面、OpenGL 加速画布、色彩管理、功能强大的笔刷引擎、非破坏性图层和蒙版、图层分组、矢量图形支持和可切换界面预设等在内的一系列特色功能。

安装 Kritaopen in new windowextra / aur：

sudo pacman -S krita

yay -S krita-git

# Gifcurry

Gifcurryopen in new window 是面向 GIF 制作者的开源、Haskell 构建的视频编辑器。

安装 Gifcurryopen in new windowaur：

yay -S gifcurry
